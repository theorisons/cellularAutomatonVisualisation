# Simulation d'Automate Cellulaire

## Présentation
Code du visualisateur d'automate cellulaire présenté dans la video **[Automates Cellulaires | Jeu de la Vie | Structures de Données](https://youtu.be/uqTgQ07sDF8 "Voir la vidéo")**.


## Version web en ReactJs

### Utilisation

* #### Web 
Le site est hébergé directement sur GitHub Pages, c'est-à-dire que le site est accessible sur ce  **[lien](https://theorisons.github.io/cellularAutomatonVisualisation/ "tester le visualisateur")**. 

* #### Local

##### Prérequis :

* Installer npm
* Installer Node.js
* Installer git
* Installer create-react-app :
`npm install -g create-react-app`

##### Déroulement :

* Cloner le repository :  
`git clone https://github.com/theorisons/cellularAutomatonVisualisation.git`   en https  
`git clone git@github.com:theorisons/cellularAutomatonVisualisation.git`   en ssh
* Se positionner dans le dossier de l'application :
`cd cellularAutomatonVisualisation`
* Installer les dépendances :
`npm install`
* Lancer l'application :
`npm start`
